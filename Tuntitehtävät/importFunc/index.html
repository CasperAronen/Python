Auditin testitulokset

Tässä on esimerkki HTML-dokumentista, joka sisältää kuvankaappaukset testien tuloksista ja esimerkin CSS-koodista käyttämästäsi fontista.

Esimerkki CSS-koodista (käytetty fontti)

Alla näkyvä CSS näyttää, miten liität Google Fonts -fontin ja miten määrittelet oman @font-face -säännön, jos toimitat fonttitiedoston.


/* Google Fonts -esimerkki */
@import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;600;800&display=swap');

body {
  font-family: 'Inter', system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;
  font-size: 16px;
  color: #111827;
}

/* @font-face -esimerkki omalle fontille (tarvitset .woff/.woff2-tiedostot) */
@font-face {
  font-family: 'OmaFontti';
  src: url('fonts/OmaFontti.woff2') format('woff2'),
       url('fonts/OmaFontti.woff') format('woff');
  font-weight: 300 800;
  font-style: normal;
  font-display: swap;
}

.header {
  font-family: 'OmaFontti', 'Inter', sans-serif;
  font-weight: 700;
}
